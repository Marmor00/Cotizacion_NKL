# Cotizador de Aluminio y Cristal

Sistema de cotización para ventanas, puertas y productos de aluminio.

## Ver Demo

Abre [mockup-cotizador.html](mockup-cotizador.html) en tu navegador para ver el mockup interactivo.

## Características

- Cotización de productos de aluminio
- Cálculo automático de materiales
- Múltiples tipos de vidrio y acabados
- Fórmulas personalizables
- Interfaz moderna y responsive

## Uso

1. Selecciona un producto
2. Configura el acabado y tipo de vidrio
3. Ingresa las medidas
4. Presiona "Calcular Cotización"

---

Desarrollado con Apps Script y HTML/CSS/JavaScript
